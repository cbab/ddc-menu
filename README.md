ddc-menu
========

### Dependency
This utility require [Node.js](http://www.nodejs.org) to be installed on your system.

### Installation and usage
````
$ git clone https://github.com/cbab/ddc-menu
$ cd ddc-menu
$ npm install
$ node menu.js
````
